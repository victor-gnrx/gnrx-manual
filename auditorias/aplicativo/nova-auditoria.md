# Como Iniciar uma Nova Auditoria

O aplicativo GNRX Auditorias permite que você crie e inicie novas auditorias diretamente do seu dispositivo móvel, seguindo um processo simples e intuitivo.

## Pré-requisitos

Antes de iniciar uma nova auditoria, certifique-se de que:

- Você está logado no aplicativo com sua conta GNRX
- Seu dispositivo possui espaço de armazenamento suficiente
- Os modelos de checklist necessários foram previamente baixados para uso offline (se aplicável)
- Você tem permissão para criar novas auditorias no sistema

## Passo a Passo para Iniciar uma Auditoria

Siga estas etapas para iniciar uma nova auditoria no aplicativo GNRX:

1. Na tela inicial, toque no botão "**Nova Auditoria**" (ou no ícone "+" no canto inferior direito)
2. O aplicativo exibirá a lista de modelos de checklist disponíveis
3. Use a barra de pesquisa no topo para encontrar um modelo específico, se necessário
4. Os modelos são organizados por categorias e podem ser filtrados por:
   - Modelos favoritos
   - Modelos recentes
   - Modelos personalizados
   - Modelos baseados em NRs
5. Toque no modelo desejado para selecioná-lo

## Preenchendo as Informações Básicas

Após selecionar o modelo, você precisará preencher as informações iniciais da auditoria:

1. **Nome da Auditoria**: Digite um nome descritivo e objetivo
   - Exemplo: "Inspeção de EPI - Setor Produção - Maio/2025"
   
2. **Local/Unidade**: Selecione a unidade onde a auditoria será realizada
   - Toque no campo para ver a lista de locais cadastrados
   - Se o aplicativo tiver permissão de localização, os locais mais próximos aparecerão primeiro

3. **Data de Realização**: A data atual é preenchida automaticamente
   - Toque para alterar, se necessário

4. **Descrição** (opcional): Adicione detalhes sobre o propósito ou contexto da auditoria
   - Este campo é útil para incluir informações específicas que não estão no modelo

5. **Setor** (se aplicável): Selecione o setor específico dentro da unidade
   - Este campo só aparecerá se a unidade selecionada tiver setores cadastrados

## Campos Adicionais Específicos

Dependendo do modelo selecionado, campos adicionais podem ser solicitados:

- **Tipo de Inspeção**: Selecione entre opções como Periódica, Extraordinária, Inicial, etc.
- **Responsável pela Área**: Indique o gestor ou responsável pelo local auditado
- **Número de Colaboradores**: Informe quantas pessoas trabalham no local inspecionado
- **Condições Climáticas**: Relevante para auditorias em ambientes externos
- **Turno**: Manhã, Tarde ou Noite, dependendo do horário da auditoria

> **NOTA**: Os campos obrigatórios são marcados com um asterisco (*) e devem ser preenchidos para prosseguir.

## Iniciando a Auditoria

Após preencher todas as informações necessárias:

1. Verifique se todos os campos obrigatórios foram preenchidos corretamente
2. Toque no botão "**Iniciar Auditoria**" na parte inferior da tela
3. O aplicativo criará a nova auditoria e abrirá a tela de preenchimento do checklist
4. A auditoria será automaticamente salva com status "Em andamento"

## Considerações sobre o Modo Offline

Se você estiver criando uma nova auditoria sem conexão com a internet (modo offline):

- Somente modelos previamente sincronizados estarão disponíveis
- A auditoria será armazenada localmente no dispositivo
- Um ícone de "nuvem com traço" indicará que a auditoria ainda não foi sincronizada
- Você deve sincronizar a auditoria assim que tiver conexão disponível

> **IMPORTANTE**: Auditorias criadas no modo offline existem apenas no seu dispositivo até serem sincronizadas. Caso desinstale o aplicativo ou saia da sua conta, essas auditorias serão permanentemente perdidas.

## Copiar uma Auditoria Existente

Para agilizar o processo, você pode iniciar uma nova auditoria com base em uma existente:

1. Na lista de auditorias, localize a auditoria que deseja usar como base
2. Deslize para a esquerda e toque no ícone de "Copiar" ou use o menu de opções
3. O aplicativo abrirá a tela de nova auditoria com os campos pré-preenchidos
4. Modifique as informações conforme necessário
5. Toque em "Iniciar Auditoria" para criar a nova versão

## Auditorias Agendadas

Se auditorias foram agendadas previamente no sistema web, elas aparecerão com um status especial:

1. Acesse a seção "Auditorias Agendadas" na tela inicial
2. Você verá a lista de auditorias programadas para realização
3. Toque na auditoria agendada que deseja iniciar
4. Os campos já estarão pré-preenchidos conforme o agendamento
5. Verifique as informações e toque em "Iniciar Auditoria"

## Solução de Problemas Comuns

### Não consigo ver todos os modelos
- Verifique sua conexão com a internet
- Atualize a lista puxando para baixo na tela
- Certifique-se de que você tem permissão para acessar os modelos

### A auditoria não inicia após tocar no botão
- Verifique se todos os campos obrigatórios foram preenchidos
- Reinicie o aplicativo e tente novamente
- Verifique se há mensagens de erro específicas

### Local não aparece na lista
- O local pode não estar cadastrado no sistema
- Você pode não ter permissão para auditar este local
- Sincronize o aplicativo para atualizar as listas de locais
