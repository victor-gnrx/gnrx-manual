# Sistema Web GNRX Auditorias

O Sistema Web GNRX Auditorias é a plataforma central para gestão completa de auditorias, inspeções e verificações de conformidade da sua empresa. Com uma interface intuitiva e recursos avançados, o sistema permite criar, configurar, analisar e gerenciar todo o ciclo de vida das auditorias em um único ambiente integrado.

## Funcionalidades Principais

### Painel de Controle (Dashboard)

O Dashboard centraliza as informações mais importantes para gestão eficiente das auditorias:

- **Visão geral de desempenho** com indicadores de conformidade por unidade, setor e período
- **Gráficos interativos** para análise de tendências e comparativos
- **Status do Setor** (Crítico, Atenção, Bom, Ótimo)
- **Status da Unidade** (Crítico, Atenção, Bom, Ótimo)
- **Quantidade de auditorias por usuário** para verificação de metas da equipe

### Gestão de Modelos de Checklist

O sistema oferece duas abordagens para criação e gestão de modelos de checklist:

#### Modelos Customizados

Crie checklists totalmente personalizados com máxima flexibilidade:

- **Editor visual intuitivo** com estrutura hierárquica de seções, itens e subitem
- **Múltiplos tipos de resposta** para diferentes necessidades:
  - Conformidade simples (Conforme/Não Conforme/Não Aplicável)
  - Coleta de Notas de conformidade (1-5, 1-10)
  - Exibir informação
  - Campos de texto livre
  - Campos numéricos
  - Campos de hora
  - Campos de data
- **Sistema de pontuação configurável** para cálculo automático de notas por seção e geral
- **Regras de criticidade** para classificação automática de não conformidades
- **Condições de visibilidade** para perguntas dinâmicas que aparecem conforme respostas anteriores
- **Personalização de cabeçalho** com campos específicos para cada tipo de auditoria
- **Importação/exportação** de modelos para compartilhamento entre unidades

<div style="text-align: center; margin: 20px 0;">
  <img src="https://example.com/imagens/editor-modelo-customizado.png" alt="Editor de modelo customizado" width="90%" />
</div>

#### Modelos NR (Normas Regulamentadoras)

Utilize modelos pré-configurados baseados nas Normas Regulamentadoras brasileiras:

- **Biblioteca completa de NRs** atualizada conforme legislação vigente
- **Estrutura fiel aos textos normativos** organizados por itens e subitens
- **Classificação automática** de infrações por tipo e gravidade
- **Potencial de multa** calculado conforme parâmetros da legislação
- **Possibilidade de personalização** para adaptar à realidade da empresa sem perder a conformidade

<div style="text-align: center; margin: 20px 0;">
  <img src="https://example.com/imagens/modelo-nr.png" alt="Exemplo de modelo baseado em NR" width="90%" />
</div>

> **DICA DE PRODUTIVIDADE**: Use Modelos NR quando precisar de conformidade direta com a legislação e Modelos Customizados quando necessitar de maior flexibilidade ou para processos internos específicos.

### Planos de Ação

O sistema oferece duas abordagens complementares para gestão de ações corretivas:

#### Plano de Ação A: Baseado em Não Conformidades

Gerenciamento direto a partir das não conformidades identificadas:

- **Criação automática** de ações para cada não conformidade
- **Classificação por criticidade** (alta, média, baixa)
- **Definição de responsáveis e prazos** com notificações automáticas
- **Monitoramento de status** (pendente, em andamento, concluída, atrasada)
- **Registro de evidências** da implementação das correções
- **Aprovação de fechamento** por responsáveis designados

<div style="text-align: center; margin: 20px 0;">
  <img src="https://example.com/imagens/plano-acao-a.png" alt="Plano de Ação baseado em não conformidades" width="90%" />
</div>

#### Plano de Ação B: Gestão Estruturada de Projetos

Abordagem avançada para tratativas que envolvem múltiplas etapas:

- **Agrupamento de não conformidades** relacionadas em um único plano
- **Definição de ações macro e micro** com estrutura hierárquica
- **Cronograma detalhado** com datas de início e fim para cada etapa
- **Monitoramento de progresso percentual** de cada ação e do plano como um todo
- **Gerenciamento de recursos** necessários para implementação
- **Acompanhamento de custos** associados às correções
- **Integração com calendário** e sistemas de notificação

<div style="text-align: center; margin: 20px 0;">
  <img src="https://example.com/imagens/plano-acao-b.png" alt="Plano de Ação estruturado como projeto" width="90%" />
</div>

> **QUANDO USAR CADA TIPO**: Utilize o Plano A para não conformidades simples que podem ser resolvidas individualmente. O Plano B é ideal para questões mais complexas que exigem múltiplas ações coordenadas ou envolvem diferentes setores da empresa.

### Relatórios e Análises

Ferramentas avançadas para análise de dados e geração de documentação:

- **Relatórios padronizados** para auditorias individuais
- **Relatórios comparativos** entre períodos, áreas ou tipos de auditoria
- **Gráficos de tendência** para acompanhamento de evolução
- **Ranking de desempenho** por unidade, departamento ou responsável
- **Status** de não conformidades para identificação visual de áreas críticas
- **Relatórios executivos** com resumo para alta gestão

### Comunicação e Notificações

Sistema integrado para manter todos os envolvidos informados:

- **Envio automático de relatórios** por email
- **Notificações de prazos** para responsáveis por ações corretivas
- **Alertas de auditorias programadas** para equipes envolvidas
- **Lembretes de tarefas pendentes** para auditores e gestores
- **Comentários e discussões** em itens específicos para alinhamento de equipes
- **Histórico de comunicações** para rastreabilidade

### Gestão de Usuários e Permissões

Controle granular de acesso às diferentes funcionalidades:

- **Perfis pré-configurados** para diferentes funções (administrador, auditor, gestor, etc.)
- **Permissões por módulo e funcionalidade** para personalização completa
- **Restrição de acesso** por unidade, departamento ou área
- **Registro de atividades** para auditoria de ações de usuários
- **Autenticação segura** com opção de integração com sistemas corporativos (LDAP, SSO)
- **Recuperação de senha** e processos de segurança

## Integração com Aplicativo Mobile

O Sistema Web GNRX Auditorias se integra perfeitamente com o aplicativo mobile:

- **Sincronização bidirecional** de dados entre plataformas
- **Disponibilização de modelos** para uso offline no aplicativo
- **Recebimento automático** de auditorias realizadas em campo (via aplicativo)
- **Gestão centralizada** de todas as auditorias independente da origem
- **Unificação de relatórios** incluindo dados de ambas plataformas
